### Prerequisites

- Node.js installed on your machine
- Yarn package manager (optional, but recommended)

### Installation

Install dependencies:
```bash
npm i
```

### Development
Run the development server:
```bash
npm run dev
```

### Building
Run the development server:
```bash
npm run build
```
